# OMU Toplumsal Katkı Scraper

OMU Mühendislik Fakültesi sitelerinden toplumsal katkı faaliyetlerini toplayan Python scraper.

## Kurulum

```bash
pip install -r requirements.txt
```

## Çalıştırma

```bash
python get_link.py
```

Çıktı olarak `education_activities.md`, `health_activities.md`, `research_activities.md` ve `social_responsibility_activities.md` dosyaları oluşturulur.
